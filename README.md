# Number Guessing Game (Java)

A console-based number guessing game built using Core Java.

## Objective
The program generates a random number between 1 and 100.
The user has 7 attempts to guess the correct number.

## Game Features
- Input validation (1–100 only)
- Feedback: Too High / Too Low
- Attempt counter
- Game Over message if attempts run out

## Concepts Used
- Classes and Objects
- Random class
- Scanner for user input
- while loop
- Conditional statements
- Boolean control logic

## How to Run
1. Compile the program
2. Run the `NumberGuessingGame` class
3. Enter guesses between 1 and 100
